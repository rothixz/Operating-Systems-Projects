Detailed Description

Problem name: Game of the rope.

(interface file)

Concept: Pedro Mariano

Logging the internal state of the problem into a file.

Defined operations:

  • file initialization
  • writing the match header at the end of the file
  • writing the game header at the end of the file
  • writing the present state as a single line at the end of the file
  • writing the game result at the end of the file
  • writing the match result at the end of the file.
Author
António Rui Borges - November 2015

Function Documentation

void createLog ( char  nFic[])

File initialization.

The function creates the logging file and writes its header. If nFic is a null pointer or a null string, the file is created under a predefined name log.

The header consists of

  • a line title
  • a blank line
  • a double line describing the meaning of the different fields of the state line.

void saveState ( char  nFic[],
FULL_STAT *  p_fSt 
)

Writing the present full state as a single line at the end of the file.

If nFic is a null pointer or a null string, the lines are appended to a file under the predefined name log.

The following layout is obeyed for the full state in a single line

  • referee state
  • coaches state (c = 0,...,C-1)
  • contestants state and strength (n = 0,..., N-1)
  • trial state (c = 0,...,C-1 and m = 0,..., M-1).
